Introduction

Esta API de Cobranza facilita la información actualizada de las cobranzas que se hicieron.

Base URL: http://localhost:8080

Esta documentación provee toda la información necesaria para trabajar con esta API.

Autenticación

Esta API no cuenta con autenticación.

Administrador Cobranza

Esta API tiene como finalidad mostrar la información de las cobranzas realizadas, puede ser filtrada por 4 campos distintos, los cuales son: cliente, equipo, vendedor y cobrador. Estos filtros harán que se facilite la visualización y busqueda de estos datos.

Example request:
curl --request GET \
    --get "http://localhost:8080/v1/cobranzas"
const url = new URL(
    "http://localhost:8080/v1/cobranzas"
);

fetch(url, {
    method: "GET",
}).then(response => response.json());
